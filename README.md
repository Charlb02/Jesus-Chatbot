# Jesus Chatbot Arabic Site

A small React/Vite project for an Arabic guided spiritual chatbot.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit content

Open:

`src/data/content.json`

You can change:
- intro text
- action button labels
- return path message
- all topics and their replies
- simple theme colors
